# PCOO---AD-2025---proyecto-C
Pensamiento computacional orientado a objetos, proyecto 
Es un gesto de biblioteca el cual te pide tus datos (Si eres profesor o estudiante, tu nombre edad y dependiendo de la anterior respuesta, tu matrícula o departamento) 
Con estos datos va a realizar una solicitud de prestamo en la cual va a marcar el libro como ocupado despues de que hagas el prestamo 
Despues te mostrará los datos de tu prestamo, esto para facilitarte a ti el acceso a un prestamo en una biblioteca


Para ejecutarlo:
1. Descargar todos los archivos 
2. Instalar el lenguaje de programacion en C++
3. Instalar de preferencia el IDE Visual Studio Code
4. Al tener los archivos, ejecutar la terminal desde la pestarña del main y ejecutar el siguiente comando (para windows):
  4.1 "g++ main_proy.cpp o- main_proy.exe", este compila el programa.
  4.2 Posteriormente ejecutar el siguiente comando para ejecutarlo
  4.3 ".\main_proy.exe"
5. Para IOs:
  5.1 "g++ -o main_proy main_proy.cpp"
  5.2 "./main_proy"
4. Para linux:
  4.1 "g++ main_proy.cpp o- main_proy.exe", este compila el programa.
  4.2 Posteriormente ejecutar el siguiente comando para ejecutarlo
  4.3 "./main_proy"
